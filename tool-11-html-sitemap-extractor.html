HTML Sitemap Extractor

Paste any HTML page and instantly extract all links — with anchor text, full URL, type (internal/external/resource), path depth, and nofollow status. Filter, sort, and export to Excel or CSV.

✓ No API Key ✓ Internal vs External ✓ Path Depth Analysis ✓ CSV & Excel Export

💡 To get a page's HTML: Open the page → Right-click → View Page Source → Ctrl+A → Ctrl+C → Paste here.

⚠️ Many sites block direct browser fetches (CORS). If fetch fails, open the URL in your browser, press Ctrl+U (View Source), select all, copy, then paste in the Paste HTML tab.
